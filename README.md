# TPS Loot Prototype

PUBG 스타일 3인칭 슈터 루팅/인터랙션 프로토타입.

진실의 원천(source of truth)은 단일 파일 **[시뮬레이터_루팅_시스템_Prototype.html](시뮬레이터_루팅_시스템_Prototype.html)** 입니다. CSS/JS/HTML 모두 인라인이며 Three.js r128을 CDN에서 로드합니다. 이 저장소는 그 파일을 편집하고 공유하기 편하도록 Vite를 dev 서버로 얹어 둔 것일 뿐입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:5173/](http://localhost:5173/) 열기 → 자동으로 프로토타입 파일로 리다이렉트됩니다.

프로토타입 HTML을 편집하면 Vite가 브라우저를 자동 리로드합니다.

## 동료에게 공유

### 옵션 1 — LAN 시연 (가장 빠름)

```bash
npm run dev -- --host
```

표시되는 Network 주소를 같은 망의 동료에게 전달.

### 옵션 2 — 정적 호스팅용 빌드

```bash
npm run build
```

`dist/` 에 `index.html` 과 `시뮬레이터_루팅_시스템_Prototype.html` 이 함께 생성됩니다. Netlify/Vercel/GitHub Pages/사내 정적 호스트 등에 그대로 업로드.

```bash
npm run preview -- --host
```

로컬에서 빌드 결과물을 LAN으로 미리보기.

### 옵션 3 — 파일만 직접 전달

프로토타입 HTML은 완전 독립형(self-contained)이므로 `시뮬레이터_루팅_시스템_Prototype.html` 한 파일만 동료에게 보내도 브라우저에서 바로 열립니다. (CDN 접속 가능한 환경 한정.)

## 인터랙션 가이드

키보드/마우스(PC) 와 게임패드(Xbox/DualSense) 둘 다 지원. 패드를 꽂거나 입력하면 화면 힌트가 자동 전환됩니다.

### 월드 — 캐릭터 조작

| 액션 | PC | Pad |
|---|---|---|
| 이동 | `W A S D` | 좌측 스틱 |
| 시점 회전 | 마우스 | 우측 스틱 |
| 달리기 | `Shift` | `RB` |
| 점프 | `Space` | `A` |
| 앉기 | `C` | — |
| 1·3인칭 전환 | `C` 토글 (앉기 무관) | — |
| 사격 | 좌클릭 | `RT` |

### 월드 — 아이템/상자 인터랙션

화면에 보이는 표시 단계 — **인지 → 접근 → 인터랙션** 3 단계로 진행:

- **3m 안에 아이템·상자**: 화면에 흰 테두리 원(`hollow dot`) 만 표시 → "여기 뭐 있다" 인지
- **1.5m 안 + 카메라가 향함**: 원이 사라지고 버튼 가이드 prompt 표시
  - 바닥 아이템: `E` (PC) / `X` (Pad) 한 번 → 즉시 픽업
  - 상자/시체: 동일 키를 **2초 꾹** 누르면 ring 이 시계방향으로 차오르며 탐색 → 인벤토리가 열리고 좌측 "주변" 영역에 컨테이너 내용물 표시
- prompt 의 키 모양으로 동작 구분
  - 흰 원 + 검은 글자 (보더 없음) = 탭(한 번 누름)
  - 흰 원 + 보더 + 상단 ▼ 삼각형 = 꾹 누르기

> 컨테이너 인터랙션 중 키를 떼면 진행 취소. 탐색 후 1.5m 안에서 멀어지면 컨테이너 영역 자동 사라짐.

### Quick Loot — 라디얼 휠

투척·회복 아이템 빠른 사용:

- **PC**: `Q` 꾹 → 휠 열림 → 마우스로 슬라이스 선택 → `Q` 떼면 사용
- **Pad**: `LB` 꾹 → 우측 스틱으로 슬라이스 선택 → `LB` 떼면 사용

### 인벤토리

`Tab` (PC) / `Y` (Pad) — 전체 인벤 열기/닫기. `Esc` (PC) / `B` (Pad) 로도 닫힘.

좌측 컬럼이 "**주변**" — 컨테이너 내용물 + 바닥 아이템 sub-section 두 개가 통합 스크롤. 둘 다 비면 컬럼 자체가 숨김. 중앙 = **배낭** + 안전 포켓. 우측 = **장비** (무기 1·2 + 헬멧/조끼/팩 + 소모품).

#### 슬롯 조작

| 액션 | PC | Pad |
|---|---|---|
| 슬롯 포커스 이동 | 마우스 호버 | D-pad / 우측 스틱 (옵션 ON 시) / 좌측 스틱 (OFF) |
| 영역 순환 (주변 ↔ 배낭 ↔ 장비) | (마우스로 직접 클릭) | `LB` / `RB` |
| 단일 픽업 / 빠른 이동 | 더블 클릭 | `A` |
| 컨텍스트 메뉴 (드롭, 분할 등) | 우클릭 | `Y` |
| 다중 선택 | `Ctrl` + 좌클릭 (여러 슬롯 선택) | — |
| 드래그 → 슬롯 / 영역 이동 | 클릭 드래그 | (해당 없음) |
| 주변 영역 한꺼번에 가져오기 | `E` 꾹 | `X` 꾹 (포커스가 주변 또는 컨테이너일 때) |
| 슬롯 아이템 바닥에 버리기 | 우클릭 → 메뉴 | `X` 꾹 (배낭/장비 슬롯 포커스 시) |

#### 인벤 열린 상태에서도 캐릭터 이동

기본 ON — `WASD` (PC) 또는 좌측 스틱 (Pad) 으로 인벤 열린 상태에서도 캐릭터가 움직임. 슬롯 네비게이션은 이때 우측 스틱 / 화살표 / 마우스로 동작. 우측 사이드패널의 "**인벤 열려도 캐릭터 이동**" 체크박스로 끄면 예전 동작 (이동 불가, 좌측 스틱이 슬롯 네비) 으로 돌아감.

#### 호환성 표시 — HUD 와 인벤 슬롯

현재 장착 무기와 호환되는 탄약/부착물을 시각적으로 알림:

- **인벤 슬롯 우상단 초록 점**: 이 아이템이 장착 무기와 호환됨
- **바닥 픽업 prompt 하단 무기 실루엣**: 포커싱한 탄약·부착물이 호환되는 무기 표시 (가로 흰 실루엣)
- **picked prompt 우측 ↑ 화살표**: 더 좋은 등급 무기/방어구

### UI 토글

- `O` — 우측 사이드패널(프로토타입 컨트롤 옵션) 토글. 숨김 시 우상단에 `[O] 컨트롤` 힌트 뱃지만 표시.
- `Tab` — 인벤토리.

### 미세 디테일

- 카메라 walk bob + idle breathing — 1인칭일 때만 자동 적용
- 사격 시 FOV 펀치 + crosshair spread 동적 확장 (이동·점프·사격 시 더 벌어짐)
- 무기 슬롯에 무기가 들어 있으면 좌하단에 등급별 색상 삼각형 + 우하단에 사용 탄약 아이콘 + 구경 + 잔탄/최대 표시
- 부착물 슬롯에 부착물 없으면 placeholder 가 흰색 반투명 실루엣으로 "장착 가능한 자리" 라는 힌트만 줌
- 슬롯 포커스/호버 시 무지개 그라데이션 ring 이 슬롯 외곽에 표시 (rarity stroke 와 별개의 시각 cue)
- 컨테이너 탐색 직후 인벤 열리면 포커스가 자동으로 컨테이너 첫 슬롯으로 이동
